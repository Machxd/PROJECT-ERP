# PROJECT-ERP
Sistema de Gestão de Inventário para Produtos Perecíveis
-Funcionalidades: Controle rigoroso de validade, registro detalhado de movimentações (entradas, saídas e desperdícios) e dashboard analítico para tomada de decisão.

-Back-end: Desenvolvido em Python nativo, utilizando http.server para o servidor e SQLite3 para persistência de dados, mantendo o projeto livre de dependências externas.

-Front-end: Interface moderna com estética Glassmorphism (HTML/CSS/JS) e visualização de dados dinâmica através da biblioteca Chart.js.

Para rodar o código:
Execute `python servidor.py`, depois: acesse`http://localhost:8080`, 
e faça o login: `admin / admin123` ou `operador / op2024`
